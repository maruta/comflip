Comment Flipper

OBS Studio でリアルタイムに画面上を流れるコメントを実現するオープンソースなツール
このサーバーは無料で利用できます
自分専用のサーバーを立てることも容易です

使い方

部屋の名前を入力してください

  • 部屋名を知っていれば誰でもコメントの投稿/閲覧が可能です
  • 偶然の重複による事故を避けたい場合，部屋名にランダムな文字列を加えて 「テスト部屋+Uhw2f3ij」のようにしてください
    最後の+以降は部屋名として表示されませんがIDとしては識別されます

のページでコメントできます

  • コメントしてほしい人にのURLを連絡してください
  • QRコードも利用できます

のページにコメントが表示されます

  • ↑のページを OBS Studio で Browser Source として取り込んでください
  • もともと透明なのでクロマキーなどの設定は不要です
  • Browser Source のオプションの Custom CSS で文字色・サイズなどをカスタマイズできます
    上書きするべき項目についてはstyle.cssを参考にしてください

補足説明
